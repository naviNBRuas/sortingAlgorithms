# Sorting Algorithms

## Introduction
This project contains information and implementation of various sorting algorithms. The algorithms here are only comparison-based sorting algorithms. Each algorithm has its own section which includes an introduction, time complexity, method, pseudocode and an animation of the sorting process.

## Algorithms Included
1. Block Sort
2. Bubble Sort
3. Cocktail Shaker Sort
4. Comb Sort
5. Cubesort
6. Cycle Sort
7. Exchange Sort
8. Gnome Sort
9. Heapsort
10. In-place Merge Sort
11. Introsort
12. Library Sort
13. Merge Sort
14. Odd-even Sort
15. Patience Sorting
16. Quicksort
17. Selection Sort
18. Shellsort
19. Smoothsort
20. Strand Sort
21. Timsort
22. Tournament Sort
23. Tree Sort

## Time Complexity
The time complexity of each algorithm is explained in detail in each algorithm's section. A table is also provided that contains the name of the algorithm, the best possible scenario, average scenario and worst scenario, memory, stability and method.

## Requirements
* Python 3

## Usage
Clone or download the repository, then run the desired algorithm file in the terminal or any Python IDE.
