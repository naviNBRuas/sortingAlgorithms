# Sorting Algorithms Project

A comprehensive study of various sorting algorithms including their implementation in code and an analysis of their time and space complexity.

## Overview

This project provides an in-depth look at the various sorting algorithms including bubble sort, insertion sort, selection sort, merge sort, quick sort and more. The project includes the implementation of these algorithms in various programming languages such as C++, Java and Python.

## Main Document

The main document is written in Quarto and is outputted as both an html and pdf file. This document provides a list of algorithms and includes each algorithm's logic, time complexity and overview.

## Performance Analysis

The project also provides a performance analysis of each algorithm in terms of its time and space complexity. This helps to determine the most efficient algorithm for a given situation.

## Comparison Based Algorithms

Please note that this project only covers comparison based algorithms.

## Conclusion

This project provides a comprehensive study of various sorting algorithms and is a useful resource for understanding the inner workings of each algorithm and their performance characteristics. Whether you're a beginner or an experienced programmer, this project has something for everyone.
